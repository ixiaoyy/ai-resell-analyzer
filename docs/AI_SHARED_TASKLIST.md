# 共享任务列表 AI_SHARED_TASKLIST

> 所有开发者和 AI 协作的任务统一在此登记。中高风险任务开工前必须先提交 `docs/claims/` 认领说明。

## 任务状态说明

| 状态 | 含义 |
|------|------|
| `todo` | 待认领 |
| `claimed` | 已认领，未开始 |
| `in_progress` | 开发中 |
| `review` | 待 Review |
| `done` | 已完成 |
| `blocked` | 阻塞中 |

## 风险等级说明

| 等级 | 含义 |
|------|------|
| A | 低风险，可直接开工 |
| B | 中风险，建议提认领说明 |
| C | 高风险，必须提认领说明 |
